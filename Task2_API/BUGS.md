# Баг-репорты по API

### API_BUG_001: Создаются объявления с полем `sellerId` выше и ниже возможного диапазона

**Приоритет:** Высокий 

**Описание:**  
При выполнении POST-запроса для создания объявления, можно создать `sellerId` выше и ниже возможного диапазона. 

**Шаги воспроизведения:**
1. Создать объявление через POST-запрос на `/api/1/item`, в `sellerId` ввести значение либо меньше 111111, либо выше 999999

**Ожидаемый результат:**  
Ошибка 400. Некорректное тело запроса

**Фактический результат:**  
Создается объявление с полем `sellerId` выше и ниже возможного диапазона

**Окружение:**  
- API Base URL: https://qa-internship.avito.com  
- OS: Windows 11 Pro 
- Инструменты: pytest + requests, Postman

---
