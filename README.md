# QA Avito - Тестовое задание

Автоматизированные API-тесты

## Структура проекта

```
Task2_API/
├── tests/
│   ├── conftest.py              # общие фикстуры
│   ├── test_create_item.py      # тесты создания объявлений
│   ├── test_get_item.py         # тесты получения объявления
│   ├── test_get_statistic.py    # тесты получения статистики
│   └── test_get_seller_items.py # тесты списка объявлений продавца
├── utils/
│   ├── api_client.py            # HTTP-клиент
│   ├── assertions.py            # вспомогательные проверки
│   └── data_generator.py        # генератор тестовых данных
├── config.py                    # настройки (URL, таймауты и т.д.)
├── pytest.ini                   # конфигурация pytest
├── requirements.txt             # зависимости
├── TESTCASES.md                 # описание тест-кейсов
└── BUGS.md                      # найденные баги
```

## Установка и запуск

### 1. Установить зависимости

```bash
cd Task2_API
pip install -r requirements.txt
```

### 2. Запустить все тесты

```bash
pytest

```

## Дополнительно

- `sellerId` генерируется в диапазоне 111111-999999 (как требует задание).
- Часть негативных тестов ожидаемо падает, они описаны в `BUGS.md`.
- Тесты независимы: каждый создаёт и удаляет свои данные через фикстуру `created_item`.
